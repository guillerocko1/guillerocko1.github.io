Tips & Tricks

  • "Optimizing Images for the Web". To optimize the picture I used the online jpeg-optimizer. It's important to uncheck the box that says "Resize...." to keep the same width and height of the original image. The compression level should be 65, which means 65%. Then, just make a click the Optimize Photo button to optimize the picture.
  • Resizing and Compressing Digital Photos and Images Online with the JPEG Optimizer Online
  • Online JavaScript/CSS/HTML Compressor This is an amazing compressor to minify your code. This is a web interface to minify JavaScript, CSS and HTML code. This tool uses UglifyJS 2, Clean-CSS and HTML Minifier. You can easly change the defaults below the textarea.
  • HTML Validator An excellent HTML validator which allow us to identify errors and warnings.
